Build command

Run:

cargo run -- build

Local preview

Run:

cargo run -- serve

Then open:

http://127.0.0.1:3000

The preview server currently does a full build once and serves the generated site/ directory. It does not rebuild automatically yet.

Output

The generated site is written into the site/ directory.
