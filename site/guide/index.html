Guide Overview

This section is for teammates who are new to the repo.

Build command

Run:

cargo run -- build

Local preview

Run:

cargo run -- serve

Then open:

http://127.0.0.1:3000

The preview server rebuilds automatically when files in docs/ or zensical.toml change, and the open browser page reloads after a successful rebuild.
