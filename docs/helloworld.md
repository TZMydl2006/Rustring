# Hello, World!
This is a simple Rust program that prints "Hello, world!" to the console. To run this code, you can create a new Rust project using `cargo new hello_world`, replace the contents of `src/main.rs` with the code below, and then run `cargo run` in your terminal.
```rust
fn main() {
    println!("Hello, world!");
}
```

When you run the program, it will output:
```rust
Hello, world!
```
This is a basic example of a Rust program, demonstrating how to define the main function and use the `println!` macro to print text to the console.
